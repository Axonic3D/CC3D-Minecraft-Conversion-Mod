To install on 3DS, place the romfs folder in this directory: [SD card root]\luma\titles\0004000000151100\. If the title ID (0004000000151100) folder doesn't exist, create it. Make sure your 3DS is modded with Luma before installing! This mod should work fine on both old and new 3DS devices.
